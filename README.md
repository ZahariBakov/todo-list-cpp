# To-Do List (C++)

Просто конзолно приложение за управление на списък със задачи. Написано на C++ за учебни цели.

## Какво може да прави:

- Добавяне на задачи
- Изтриване на задачи
- Извеждане на текущи задачи
- Запазване на задачите във файл

## Как да компилираш:

```bash
g++ -g main.cpp -o todo.exe
./todo.exe
